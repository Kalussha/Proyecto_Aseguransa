═══════════════════════════════════════════════════════════════
    GUÍA PARA CREAR EJECUTABLE STANDALONE (.EXE)
═══════════════════════════════════════════════════════════════

OPCIÓN 1: EJECUTABLE ÚNICO (UN SOLO ARCHIVO .EXE)
--------------------------------------------------
✓ Archivo: CREAR_EJECUTABLE.bat
✓ Resultado: UN solo archivo .exe
✓ Ventajas: Fácil de distribuir (un solo archivo)
✓ Desventajas: Inicia un poco más lento, pesa más (50-100 MB)

CÓMO USAR:
1. Doble clic en "CREAR_EJECUTABLE.bat"
2. Espera 5-10 minutos
3. El archivo "GestionPolizas.exe" estará en la carpeta "dist"
4. Copia ese archivo a cualquier PC y ejecútalo


OPCIÓN 2: VERSION PORTABLE (CARPETA CON EJECUTABLE)
---------------------------------------------------
✓ Archivo: CREAR_EJECUTABLE_AVANZADO.bat
✓ Resultado: Carpeta con ejecutable y archivos
✓ Ventajas: Inicia más rápido, mejor rendimiento
✓ Desventajas: Son varios archivos en una carpeta

CÓMO USAR:
1. Doble clic en "CREAR_EJECUTABLE_AVANZADO.bat"
2. Espera 5-10 minutos
3. La carpeta "GestionPolizas_Portable" estará en "dist"
4. Comprime la carpeta en ZIP
5. Distribuye el ZIP

═══════════════════════════════════════════════════════════════
    SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════

Error: "Python no está instalado"
→ Instala Python primero en TU PC de desarrollo
→ Solo necesitas Python para CREAR el ejecutable

Error: "No se puede crear el ejecutable"
→ Ejecuta primero INSTALAR.bat
→ Verifica que tienes espacio en disco (necesitas ~500 MB libres)

El .exe pesa mucho (50-100 MB):
→ Es normal, incluye Python completo y todas las librerías
→ No se puede reducir significativamente sin comprometer funcionalidad

El .exe inicia lento:
→ Usa la Opción 2 (portable) en su lugar
→ O espera, solo el primer inicio es lento
