┌───────────────────────────────────────────────────────────┐
│  ❓ SI ERES DESARROLLADOR O TIENES EL CÓDIGO FUENTE       │
└───────────────────────────────────────────────────────────┘

1️⃣ Primera vez (configuración inicial):
   → Ejecuta: INSTALAR.bat
   
2️⃣ Para probar la aplicación:
   → Ejecuta: EJECUTAR.bat

3️⃣ Para crear ejecutable (.exe) que NO requiera instalar nada:
   
   ⭐ PRIMERO (MUY IMPORTANTE):
   → Ejecuta: INSTALAR_PYINSTALLER.bat
   → Esto evita el error "pyinstaller no se reconoce"
   
   LUEGO, elige una opción:
   
   🟢 RECOMENDADO - Script simple y robusto:
   → Ejecuta: CREAR_EXE_SIMPLE.bat
   → Resultado: dist\GestionPolizas.exe (50-100 MB)
   
   🔵 OPCIÓN A - Archivo único (más fácil de distribuir):
   → Ejecuta: CREAR_EJECUTABLE.bat
   → Resultado: dist\GestionPolizas.exe (50-100 MB)
   
   🟣 OPCIÓN B - Versión portable (mejor rendimiento):
   → Ejecuta: CREAR_EJECUTABLE_AVANZADO.bat
   → Resultado: dist\GestionPolizasPortable\ (carpeta)

   ⚠️ IMPORTANTE: Ejecuta como Administrador (clic derecho)

4️⃣ ¿Problemas al crear el .exe?
   → Lee: SOLUCION_PROBLEMAS_EXE.txt (NUEVA GUÍA)
   → O lee: COMO_CREAR_EXE.txt

                    FLUJO DE TRABAJO TÍPICO
═══════════════════════════════════════════════════════════════

DESARROLLADOR:
1. INSTALAR_PYINSTALLER.bat (solo una vez) ⭐ NUEVO
3. EJECUTAR.bat (para probar)
4. Hacer cambios al código si es necesario
5. CREAR_EJECUTABLE.bat (cuando esté listo)
6. CREAR_EJECUTABLE.bat (cuando esté listo)
5. Distribuir el .exe de la carpeta "dist"

USUARIO FINAL:
1. Recibir el .exe o carpeta portable
2. Doble clic en el ejecutable
3. ¡Usar la aplicación!
